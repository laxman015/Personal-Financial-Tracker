## Run the app
From this folder:
```bash
python app.py
```
Open the shown URL (usually `http://127.0.0.1:5000`).
## Create your account
1. Go to `Register`
2. Create a username + password
3. Log in
4. Add transactions and view balance
## Notes
- The database file `finance_data.db` is created automatically on startup.
- Amounts must be whole numbers (no floating/decimal values).
- The balance starts from `0` and then sums your Income/Expense transactions
